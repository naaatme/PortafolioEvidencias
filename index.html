Bienvenido

Este portafolio presenta los conocimientos adquiridos durante el cuatrimestre en la materia de Desarrollo Web Orientado a Servicios.

Se aborda el uso, desarrollo e implementación de APIs, así como su integración en aplicaciones web modernas.


Objetivo del proyecto: Aplicar de forma práctica los conceptos y habilidades aprendidas durante el curso.

Unidades del Curso

Introducción al Desarrollo Web Orientado a Servicios

Conceptos fundamentales sobre servicios web, nube y arquitectura orientada a servicios.

Implementación de APIs

Consumo de APIs existentes y comunicación entre aplicaciones.

Desarrollo de APIs

Diseño y creación de APIs propias siguiendo buenas prácticas.

Implementación Final

Integración completa del proyecto y validación de funcionalidades.
